# VELVET Transcrição

> Edite pelo texto como no Premiere, com transcrição por IA na SUA GPU — sem nuvem, sem limite de horas.

Plugin de **Integração de Fluxo de Trabalho** para o **DaVinci Resolve Studio**. Nativo, 100% offline, em português. Parte do ecossistema **VELVET** by Paulocodex.

## O que faz

- Transcrição GPU local (Whisper) — sem custo por minuto
- Editar por texto: excluir corta o vídeo (ripple)
- Excluir pausa/muleta, localizar/substituir
- Importar/exportar SRT · legenda nativa
- Todas as atualizações futuras

## Requisitos

- **DaVinci Resolve Studio** (as Integrações de Fluxo de Trabalho existem só no Studio; a versão gratuita não abre plugins WI).
- Windows 10/11. O instalador já traz tudo (IA local + ffmpeg) — **você não precisa instalar Python**.

## Baixar, instalar e usar

1. Baixe o **VELVET-Transcricao-Setup.zip** na aba [Releases](../../releases/latest) e extraia.
2. **Feche o DaVinci Resolve.**
3. Dê dois cliques em **INSTALAR.bat**.
4. Abra o DaVinci Studio → **Área de trabalho › Integrações de Fluxo de Trabalho › VELVET Transcrição**.
5. Na barra de baixo, clique em **Ativar** e cole a sua chave.
