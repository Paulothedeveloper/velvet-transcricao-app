## Requisitos

- **DaVinci Resolve Studio** (Integrações de Fluxo de Trabalho só existem no Studio).
- Windows 10/11. O instalador traz tudo (IA local + ffmpeg) — **não precisa instalar Python**.

## Baixar, instalar e usar

1. Baixe o instalador na aba [Releases](../../releases/latest) e extraia.
2. **Feche o DaVinci Resolve.**
3. Dois cliques em **INSTALAR.bat**.
4. DaVinci Studio → **Área de trabalho › Integrações de Fluxo de Trabalho › VELVET Transcrição** → **Ativar** → cole a chave.
